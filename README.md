# Egg-Timer
An application built to test our knowledge with the modules learned during the IOS Development Boot Camp. It includes:

	- Swift Collection Types (Dictionaries)
	- The Swift Timer API
	- Conditional Statements (If-Else / Switch)
	- Functions with Outputs
	- How To Use THe ProgressView
